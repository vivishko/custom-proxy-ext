## Overview

This release improves proxy management flow, makes proxy toggles take effect immediately on the current page, and includes a structural refactor of the extension codebase.

## New

- None.

## Improvements

- After saving a proxy, the UI navigates to the proxies table so the new entry is visible right away.
- When toggling proxy modes, the current tab reloads (for http/https pages) to reflect the updated routing immediately.
- Extension codebase refactored: popup/background split into focused modules, shared storage layer added, repo layout simplified.
