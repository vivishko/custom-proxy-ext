## Overview

This major release turns the extension popup into a fuller proxy management workspace. It adds a refreshed interface, settings, theme control, onboarding, better rule discovery, safer imports, and stronger release/test tooling around the extension package.

## New

- Settings screen for technical preferences and popup-level controls.
- Manual light/dark theme switch persisted across popup sessions.
- First-run onboarding flow with setup guidance and a shortcut back to settings.
- Seeded development command for deterministic manual verification.

## Improvements

- Redesigned popup styling with clearer visual hierarchy and table controls.
- Popup markup split into reusable templates for easier maintenance.
- Proxies are shown before rules to match the setup flow.
- Rules table now supports search, sorting, filters, and pagination.
- Proxies table now supports pagination.
- Shared i18n strings scaffold added for UI text management.
- Main-only release tagging helper and stronger CI/package checks added for safer releases.

## Fixes & stability

- Duplicate rule warning and edit flows prevent accidental duplicate rules.
- Rule imports and proxy imports now handle duplicates more safely.
- Proxy deletion warns about dependent rules and converts affected rules safely.
- Proxy explanation/status rows are easier to scan with active/inactive coloring.
