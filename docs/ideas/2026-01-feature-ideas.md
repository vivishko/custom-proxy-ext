# Feature ideas

---

## LOW

- L1 - сделать "enable proxy" не галочкой, а тумблером
- L2 - строку с объяснением прокси пометить цветом для понятности
- L3 - при добавлении/редактировании правила вручную добавить предупреждение о существующем рулсе для домена

## MEDIUM

- M1 - сделать пагинацию по 10 строк в таблице рулсов
- M2 - сделать пагинацию по 10 строк в таблице проксей
- M3 - добавить сортировку и фильтры для рулсов - добавленные недавно, по алфавиту, по прокси сеттингс
- M4 - добавить возможность редактировать правила - значок карандаша рядом с доменом в таблице + при редактировании правила если домен уже существует выдать предупреждение
- M5 - сделать поиск по рулсам

## HIGH

- H1 - главное окно - одна кнопка с включением выключением - рулсы и прокси в отдельные экраны, создание прокси тоже отдельным экраном (на странице проксей просто большая кнопка "+")
- H2 - при удалении прокси сделать окно предупреждения о связанных рулсах: с возможностью выкачать список рулсов, принадлежащих проксе, или предложением перевести все домены этой прокси на другую проксю
- H3 - при импорте рулсов выводить окна предупреждений о наличиях одинаковых записей с предложением переписать или скипнуть запись или вообще отменить импорт
- H4 - при импорте проксей добавить окно предупреждения о наличии одинаковой записи с предложением переписать или скипнуть запись или вообще отменить импорт
- H5 - изменить ui в принципе - другой стиль
