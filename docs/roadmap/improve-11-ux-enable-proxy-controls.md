# IMPROVE-11. UX: Enable Proxy controls

## Goal
Сделать управление режимами прокси понятным: глобальный режим и режим "только для этой страницы" должны быть очевидными, взаимоисключающими и легко объяснимыми.

## Value
Пользователь сразу понимает, какой режим активен и где прокси применяется.
