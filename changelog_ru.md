# Changelog

Все заметки об изменениях в этом проекте будут задокументированы в этом файле.

Формат основывается на [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [Unreleased]

### Added

- Группы прокси с балансировкой нагрузки и fallback-механизмом.
- Возможность тестировать доступность прокси прямо из popup-интерфейса.
- Индикаторы состояния прокси: успешность подключения, время последней проверки.
- Наборы предустановленных правил для популярных доменов (Google, AWS, соцсети).
- Быстрый переключатель в панели инструментов для мгновенного включения/выключения глобального прокси.

### Changed

- Улучшенный разбор доменов с поддержкой IDN.
- Переработанный интерфейс для лучшей читаемости на узких экранах.

### Fixed

- Более стабильное получение URL активной вкладки при медленной загрузке страниц.
- Исправления редких случаев генерации избыточных правил в PAC-скрипте.

### Security

- Шифрование паролей прокси с использованием платформенных API Chrome.

- Возможность включения функционала логирования для дебаггинга проблем

## [1.1.0] - 2025-11-14

### Добавлено

- Прокси
  - Добавление и удаление настроек прокси вручную
  - Экспорт и импорт данных списка прокси
- Глобальное поведение
  - Возможность включать глобальное проксирование для всех запросов в браузере + выбор конкретного варианта прокси
  - Возможность включать проксирование запросов только на одном конкретном сайте
- Правила
  - Фиксирование определенного поведения за доменом
    - _\<proxy-name\>_ - при подключении к сайту всегда используется конкретный заданный прокси из списка
    - _DIRECT_ - прямое подключение
    - _RANDOM_PROXY_ - при подключении к сайту всегда используется рандомный прокси из списка существующих
  - Добавление, изменение и удаление правила
  - Задать из глобального поведения
